## 技术性内容
这个网站的i18n很烂.  
对于任何带'key'参数的元素, key就是中文.  
可以在/js/lang.js里头看到一个arrLang表, 对应语言翻译可以往里面写.  
对于'key_?'参数, 例如'key_en_us'参数就是英翻.  
(优先识别'key_?'内文本)
